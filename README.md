# 💼 Smart Resume Analyzer

An NLP-powered web application that compares a resume with a job description and calculates a match score using TF-IDF and cosine similarity.

## 🚀 Live Demo
👉 https://smart-resume-analyzer-zl4sxphcn7b4rjq9j4basq.streamlit.app/

## 🛠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- PyPDF2
- NLP (TF-IDF)

## 📌 Features
- Upload resume (PDF)
- Paste job description
- Calculates similarity score
- Shows missing keywords
- Interactive UI with progress bar
